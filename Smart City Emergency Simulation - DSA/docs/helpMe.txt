================================================================================
USER INPUT GUIDE & FORMATTING RULES
================================================================================

[CRITICAL RULE]
When entering location names, you MUST replace all spaces with hyphens (-).

Reason: The system reads inputs separated by spaces. If you type "Bus Stop 1", 
the program treats "Bus", "Stop", and "1" as three separate inputs, which will 
cause an error.

--------------------------------------------------------------------------------
CORRECT vs INCORRECT EXAMPLES
--------------------------------------------------------------------------------

(X) Incorrect        (O) Correct
----------------     ----------------
Bus Stop 1           bus-stop-1
Police Station       police-station
Town Hall            town-hall
Fire Station         fire-station
